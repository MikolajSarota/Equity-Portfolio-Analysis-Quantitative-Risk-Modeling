A Python project for portfolio risk analysis and Monte Carlo simulation with results stored in PostgreSQL.

Features:
Historical VaR and CVaR
Monte Carlo VaR / CVaR
CAPM alpha & beta
Annual return, volatility, Sharpe ratio
Benchmark comparison (S&P 500)
Results saved to a SQL database

Tech Stack:
Python, NumPy, Pandas, yfinance, Matplotlib, Seaborn, SQLAlchemy, PostgreSQL
